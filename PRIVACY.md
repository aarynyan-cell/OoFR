# OoFR 法语磁带隐私说明

最后更新：2026-07-04

OoFR 法语磁带是一个本地优先的法语学习工具。当前版本没有账号系统、服务器数据库或内置统计分析。

## 本地保存的数据

应用会在你的浏览器或插件本地空间保存以下数据：

- 专辑、磁带、段落条目和练习次数。
- 单词本、单词状态、音标、中文释义和自定义词本。
- 播放设置，例如声音、语速、重复次数和间隔。

网页/PWA 版使用浏览器 `localStorage`。插件版使用 `chrome.storage.local` 保存完整学习数据，并使用 `chrome.storage.sync` 同步少量播放设置。

## 数据导入与导出

只有当你主动点击导入或导出时，应用才会读取或生成 JSON 文件。导出的文件可能包含你的学习文本、单词本和播放设置，请妥善保存，不要上传到公共仓库或公开分享链接。

## 语音朗读

应用使用浏览器和操作系统提供的 Web Speech API 朗读文本。部分在线自然语音可能由浏览器或操作系统供应商处理朗读内容；具体处理方式取决于你使用的浏览器、系统和语音服务。
