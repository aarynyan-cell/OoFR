# OoFR 法语发音练习器

一个纯前端的法语发音练习网页。打开 `index.html` 后，可以输入法语文本，按句子播放，也可以点击单词单独播放。

## 使用

直接用浏览器打开：

```text
index.html
```

推荐使用 Chrome、Edge 或 Safari，并在系统里安装可用的法语语音。网页使用浏览器自带的 Web Speech API，不需要后端、API key 或联网服务。

## 当前能力

- 自动分句
- 点击句子播放
- 点击单词播放
- 选择法语声音
- 调整语速
- 设置重复次数和间隔
- 自动保存文本和播放设置

## 后续可升级

- 接入更自然的 TTS 服务
- 增加 IPA / 音标显示
- 增加 liaison、重音和易错音提示
- 做成 PWA，放到手机桌面
